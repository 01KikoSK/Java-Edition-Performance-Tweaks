Java Edition Performance Tweaks

Allocate More RAM

Minecraft can benefit from having more RAM allocated to it. The default allocation is often insufficient, especially for modded gameplay or large worlds.

How to:

  1. Open the Minecraft Launcher.
  2. Go to "Installations."
  3. Select your desired installation and click "Edit."
  4. Click "More Options."
  5. In the "JVM Arguments" field, modify the -Xmx value. For example, -Xmx8G allocates 8 gigabytes of RAM.
  6. Save the changes.

Warning: Do not allocate more RAM than your system has available. Allocating too much can lead to system instability.

Allocate roughly half of your available ram. For example, if you have 16GB of ram, allocate 8.

Optimize Video Settings

Adjusting video settings can significantly impact performance.

Setting Impact Recommendation
Render Distance High Reduce to 8-12 chunks for better performance.
Graphics Medium Set to "Fast" instead of "Fancy."
Smooth Lighting Medium Turn off or set to "Minimum."
Particles Medium Set to "Minimal" or "Decreased."
Entity Shadows Medium Turn off.
Fullscreen Low Fullscreen mode can sometimes improve performance.
VSync Low Turning off VSync may increase FPS, but can cause screen tearing.
Use Performance Mods

Performance mods can significantly improve FPS and reduce lag.

Recommended Mods:

  • Sodium (Fabric): A modern rendering engine that significantly improves FPS.
  • Lithium (Fabric): Optimizes game logic without changing gameplay.
  • Starlight (Fabric): Optimizes the lighting engine.
  • OptiFine (Forge/Standalone): A popular optimization mod with many settings. (Sodium is generally better)
  • FerriteCore (Fabric): Reduces RAM usage.

For best results, use Fabric and Sodium, Lithium, and Starlight. Forge is an older mod loader, and Optifine, while useful, is less optimized than the Fabric alternatives.

Update Java and Graphics Drivers

Ensure you are using the latest version of Java and your graphics drivers.

  • Java: Download the latest Java SE Development Kit (JDK) from Oracle or Adoptium.
  • Graphics Drivers: Update your drivers from the manufacturer's website (NVIDIA, AMD, Intel).
Reduce Background Processes

Close unnecessary applications and background processes to free up system resources.

  • Use Task Manager (Windows) or Activity Monitor (macOS) to identify and close resource-intensive applications.
  • Disable startup programs that you don't need.
